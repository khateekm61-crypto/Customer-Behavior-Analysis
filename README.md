# Customer Behavior Analysis

## Project Overview
This project focuses on analyzing customer purchasing behavior using transaction data. The analysis helps identify customer patterns, spending habits, product preferences, and customer segmentation.

---

## Objectives
- Analyze customer purchasing behavior
- Perform data cleaning and preprocessing
- Create visualizations for better insights
- Identify customer segments using K-Means Clustering
- Generate business insights and recommendations

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Dataset Information
The dataset contains customer transaction details such as:
- Customer ID
- Purchase Date
- Product Category
- Product Price
- Quantity
- Payment Method
- Customer Age
- Gender
- Returns
- Churn Status

---

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Customer Segmentation using K-Means
6. Business Insights and Recommendations

---

## Visualizations Included
- Age Distribution
- Gender Distribution
- Product Category Distribution
- Purchase Amount Distribution
- Customer Segmentation Scatter Plot

---

## Key Insights
- Customer purchasing behavior varies across different customer groups.
- Product categories show balanced customer interest.
- High-value customers contribute significantly to sales.
- Customer segmentation helps identify spending patterns.
- Churn analysis helps identify retention risks.
